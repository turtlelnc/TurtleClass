# Test Strategy

## 测试层级

### 单元测试

- Core 状态机
- 事件验证
- 撤销
- 重算
- 哈希与签名封装
- 序列化

### 集成测试

- Core + Storage
- Storage + Recovery
- Server + Sync
- Auth + Device

### 端到端测试

- 两客户端
- 服务端停机恢复
- 冲突处理
- 客户端重建

### 故障测试

- 文件截断
- 快照损坏
- 重复事件
- 乱序事件
- 时钟偏移
- 网络中断
- 备份缺失

### 随机与模糊测试

对积分状态机和事件解析进行随机测试。

## 完成门槛

- 所有必需测试通过
- 无崩溃
- 无未处理异常
- 无数据静默丢失
- 失败路径有明确错误
