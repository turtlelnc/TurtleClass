# Deployment Notes

## 服务端环境

- Tom 的 MacBook Air
- macOS
- 自动启动
- Cloudflare Tunnel
- 每周固定维护一天
- 允许临时停服

## 启动流程

1. 检查配置
2. 验证凭据权限
3. 检查事件文件
4. 校验快照
5. 必要时重建
6. 启动本地服务
7. 启动 Tunnel
8. 运行健康检查

## 停服流程

1. 进入维护模式
2. 停止接受新管理操作
3. 完成当前事务
4. 校验事件链
5. 创建快照
6. 创建备份
7. 关闭服务
8. 关闭 Tunnel

正式命令在 Phase 2 实现后补充。
